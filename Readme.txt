Running the system:-
All code is written in Java 8.

To compile file:-
javac *.java
OR
javac Processor.java Mapper.java FileProcessor.java Analyzer.java Sorter.java

To run:-
java Processor transactions-person-A.csv transactions-person-B.csv

The file Processor.java contains main function.
If file is not in the same folder or package please provide file path in arguments.
